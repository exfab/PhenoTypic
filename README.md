# PhenoScope
![Development Status](https://img.shields.io/badge/status-Pre--Alpha-red)

An image processing framework created by NSF Ex-FAB BioFoundry
---
## Overview
This framework aims to provide a toolkit & modular framework that streamlines the process of creating phenotyping analysis pipelines. 
By following a structured data flow, developers can create their own versions of computer vision operations and easily integrate it
with other established protocols without having to create entirely programs every time.

## Installation
```  
git clone https://github.com/Xander-git/PhenoScope.git
cd PhenoScope
pip install -e .
```
