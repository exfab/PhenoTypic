Workflow
----------------

A typical workflow in Phenotypic involves the following steps:
   1. Input Data
     - Utilize data formats such as TIFF, PNG, and JPEG for image input.
   2. Preprocessing
     - Apply techniques like normalization, noise reduction, and filtering to prepare images for analysis.
   3. Feature Extraction
     - Extract relevant features using methods such as area, integrated intensity, and morphological characteristics.
